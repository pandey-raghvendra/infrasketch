Bug reports

If you've found a bug or a Terraform resource type that isn't supported, the best way to report it is by opening an issue on our GitHub repository. Please include the Terraform code that caused the issue (with any sensitive values removed) so we can reproduce and fix it.

Feature requests

We're actively building new features for InfraSketch. If there's something you'd like to see — a new cloud provider, a new input format, better layout options — let us know via GitHub Issues or email. Popular requests get prioritized.

Frequently asked questions

Is InfraSketch really free?

Yes, completely free. The tool is open source, runs entirely in your browser, and is kept intentionally lightweight.

Is my code safe?

Absolutely. Your Terraform code never leaves your browser. There is no server-side processing, no data transmission, and no storage. You can verify this yourself — the source code is publicly available on GitHub.

Which resource types are supported?

InfraSketch currently supports 25+ AWS resource types including VPC, Subnets, EC2, EKS, ECS, Lambda, RDS, DynamoDB, ElastiCache, S3, ALB/NLB, CloudFront, Route 53, SQS, SNS, IAM, Security Groups, WAF, KMS, CloudWatch, NAT Gateway, Internet Gateway, and more. We're adding new types regularly.

Can I use the diagrams commercially?

Yes. Diagrams you generate with InfraSketch are yours to use however you like — in documentation, presentations, proposals, or any other purpose.
