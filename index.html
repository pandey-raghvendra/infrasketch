Your IaC code,
visualized instantly.

InfraSketch is a free, browser-based infrastructure diagram generator. Paste Terraform HCL, CloudFormation YAML/JSON, CDK synth output, Terragrunt units, or Docker Compose YAML — get a clean, exportable architecture diagram in seconds. No account. No cloud credentials. Nothing leaves your browser.

Most teams document infrastructure manually in Lucidchart or draw.io, redrawing the same boxes every time something changes. InfraSketch reads your actual code and renders the diagram automatically, keeping documentation in sync with what is deployed.

Supports 75+ resource types across AWS, Azure, and GCP. VPCs, subnets, compute clusters, databases, storage, load balancers, serverless, queues, and more — automatically grouped into logical zones with clean connection routing.

How it works

Three steps from infrastructure code to a shareable architecture diagram.

01

Paste your IaC code

Copy Terraform HCL, CloudFormation YAML/JSON, CDK synth output, Terragrunt units, or docker-compose.yml into the editor. No reformatting needed — InfraSketch parses it as-is, including partial files and module blocks.

02

Auto-detect architecture

The parser reads VPC boundaries, subnet membership (subnet_id, subnet_ids), and resource references to build a containment graph automatically.

03

Export and share

Download as PNG, SVG, draw.io XML, or Mermaid code. Share a URL that encodes your diagram state — no server, no storage, no account required.

Everything you need

Built for engineers who want accurate diagrams without the overhead of visual tools.

No credentials needed

Unlike Cloudcraft or Hava, InfraSketch never touches your cloud account. Paste Terraform HCL, CloudFormation YAML/JSON, CDK synth output, Terragrunt units, or Docker Compose — everything runs in your browser.

Official cloud icons

Generates diagrams using official AWS, Azure, and GCP icons. Professional enough for documentation, presentations, and architecture reviews.

Export & import draw.io

Download as PNG, SVG, or draw.io XML. A visual preview shows your diagram before download. Or upload any SVG and convert it to native draw.io format with editable AWS/Azure stencils.

100% private

Your code never leaves your browser. No server processing, no storage, no telemetry. Fully client-side.

Full module expansion

Upload a ZIP of your Terraform project to expand local modules inline, or toggle Auto-fetch registry modules to pull public modules like terraform-aws-modules/vpc/aws from the Terraform Registry and render every resource inside.

Clean connection routing

Skips structural wiring references and only draws meaningful flow connections — no spaghetti arrows from every Terraform resource reference.

Interactive editor

Click Edit to drag any node to a new position. Connection arrows update live. Click Reset Layout to restore the original auto-layout at any time.
