What is InfraSketch?

InfraSketch is a free, browser-based infrastructure diagram generator. Paste Terraform HCL, a plan JSON, Terragrunt units, or Docker Compose YAML and get a clean, exportable architecture diagram in seconds — no account, no cloud credentials, no data ever leaving your browser.

Most teams document infrastructure manually in Lucidchart or draw.io, redrawing the same boxes every time a resource changes. InfraSketch reads your actual code and renders the diagram automatically, so your documentation stays in sync with what is deployed.

The tool supports 75+ resource types across AWS, Azure, and GCP — VPCs, subnets, compute clusters, databases, storage, load balancers, serverless functions, messaging queues, and more. Detected relationships are drawn as edges and resources are grouped into logical zones so the diagram is readable without extra layout work. Use the interactive editor to drag nodes into any arrangement you prefer.

Diagrams export as PNG for documentation, SVG for vector editing, or draw.io XML for further customization in diagrams.net. You can also go the other way — upload any SVG architecture diagram and convert it to native draw.io XML with editable shapes and official AWS/Azure stencils. A shareable URL encodes your code in the fragment so you can send a link directly to a colleague without any server-side storage.

How it works

Three steps from infrastructure code to a shareable architecture diagram.

STEP 01

Paste your IaC code

Copy Terraform HCL, CloudFormation YAML/JSON, CDK synth output, Terragrunt units, or docker-compose.yml into the editor. No reformatting needed — InfraSketch parses it as-is, including partial files and module blocks.

›
STEP 02

Auto-detect architecture

The parser reads VPC boundaries, subnet membership (subnet_id, subnet_ids), and resource references to build a containment graph — separating structural wiring from actual traffic flow.

›
STEP 03

Export and share

Download as PNG for presentations, SVG for docs and READMEs, or draw.io XML to keep editing in diagrams.net. Or upload any SVG diagram and convert it to draw.io — all client-side, nothing uploaded.

Everything you need

Built for engineers who want diagrams without the overhead of visual tools.

No credentials needed

Unlike Cloudcraft or Hava, InfraSketch never touches your cloud account. Paste Terraform HCL, CloudFormation YAML/JSON, CDK synth output, Terragrunt units, or Docker Compose — everything runs in your browser.

Official cloud icons

Generates diagrams using official AWS, Azure, and GCP icons. Professional enough for documentation, presentations, and architecture reviews.

Export & import draw.io

Download as PNG, SVG, or draw.io XML. A visual preview shows your diagram before download. Or upload any SVG and convert it to native draw.io format with editable AWS/Azure stencils.

100% private

Your code never leaves your browser. No server processing, no storage, no telemetry. Fully client-side.

Full module expansion

Upload a ZIP of your Terraform project to expand local modules inline, or toggle Auto-fetch registry modules to pull public modules like terraform-aws-modules/vpc/aws from the Terraform Registry and render every resource inside.

Clean connection routing

Skips structural wiring references and only draws meaningful flow connections — no spaghetti arrows from every Terraform resource reference.

Interactive editor

Click Edit to drag any node to a new position. Connection arrows update live. Click Reset Layout to restore the original auto-layout at any time.
